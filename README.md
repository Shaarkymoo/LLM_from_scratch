This is my attempt to create AI from scratch without using any external libraries.

## Steps in this project

1. Create a very basic Neural Network (3 layers) that works, irrespective of result
2. implement batch processing
3. add more layers and increase accuracy to above 95%
4. optimize using cuda
5. attention
6. transformer
7. functioning chatbot

**probably will add more steps as complexity in my head increases.

## Mini-Model features
3 layers in Neural network:
* layer 1 taking MNIST inputs (784 neurons)
* layer 2 having 16 neurons
* layer 3 having 10 neurons, taken as output
